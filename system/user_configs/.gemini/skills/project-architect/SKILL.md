# Project Architect Skill

This skill ensures that new features or applications are properly designed before any code is written.

## Architecture Workflow

When the user asks you to create a new application or complex feature:

1. **Enter Plan Mode**: Call the `enter_plan_mode` tool immediately.
2. **Draft the Design**: In plan mode, write a comprehensive design document detailing the architecture, tech stack, file structure, and implementation steps.
3. **Wait for Approval**: Present the plan to the user and wait for their explicit approval.
4. **Execute**: Once approved, follow the plan step-by-step to implement the application.
